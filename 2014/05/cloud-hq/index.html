Cloud HQ –  How I (automatically) sync all my files across the web.

What it is

Simply – Cloud HQ is an online service that allows you to sync your files across multiple file storing services.

Supported Services

Cloud HQ Supported Services as of 23/5/14

Cloud HQ currently supports the following syncing services:

  • Google Drive
  • Gmail
  • Dropbox
  • Box
  • Evernote
  • Onedrive
  • Onedrive Pro
  • Share Point
  • WebDAV
  • Basecamp Classic
  • Basecamp
  • Sugar Sync
  • Egnyte
  • Yandex Disk
  • Microsoft Onenote (maybe soon)
  • Salesforce (maybe soon)

Chrome Extension

I was incredibly surprised by how helpful the chrome extension is.

The extension not only  allows you to follow what files it has just synced, but in google drive  it also allows you to open a file from any of your connected services.
